# 🎮 Glitch Market - E-commerce de Videojuegos


**Glitch Market** es una aplicación web desarrollada con **React.js** para la carrera de Desarrollo de Aplicaciones en **Coderhouse**. 
Se trata de una tienda online de videojuegos con estética inspirada en las interfaces modernas de consolas (Xbox/Steam), enfocada en una navegación fluida y una experiencia de usuario inmersiva.

## 🛠️ Herramientas
- **React 18** (Vite)
- **React Router Dom v6** (Navegación)
- **CSS3** (Flexbox, Grid, Glassmorphism)
